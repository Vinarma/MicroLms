# MicroLms
A Learning Management System built using React, Node.js, Express.js, and MongoDB.

## Features

- User Authentication
- Course Management
- Student Enrollment
- Progress Tracking
- Dashboard
- Responsive Design

## Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
